# Brillo Atlántico — Web de limpieza (Las Palmas)

Sitio web estático para una empresa de limpieza en Las Palmas de Gran Canaria.

## Ver en local

Abre `index.html` en el navegador o ejecuta:

```bash
cd limpieza-las-palmas
python3 -m http.server 8080
```

Luego visita http://localhost:8080

## Personalizar

1. **Teléfono y WhatsApp**: sustituye `600 000 000` y `34600000000` en `index.html` y `script.js`.
2. **Email**: cambia `info@brilloatlantico.es`.
3. **Nombre comercial**: busca "Brillo Atlántico" y reemplázalo por el nombre real.
4. **Fotos**: en `styles.css`, cambia las URLs de Unsplash por fotos propias.

## Publicar

Sube la carpeta a cualquier hosting estático (Netlify, Vercel, GitHub Pages, tu hosting web).
